JoJo Tetris Game

A Tetris game with custom textures and level selection

About the Game

JoJo Tetris is a fun and engaging version of the classic Tetris game, featuring custom textures and a level selection system. The game is built using Python with the Kivy framework, making it compatible with Android devices.

Features:

  • Custom textures for all Tetris shapes
  • Level selection (1-10) affecting game speed
  • Classic Tetris gameplay mechanics
  • Score tracking and line clearing
  • Mobile-friendly controls

How to Play:

Controls:

  • Left/Right Arrow: Move piece horizontally
  • Down Arrow: Move piece down faster
  • Up Arrow: Rotate piece
  • Space: Drop piece to bottom
  • R: Return to level selection (after game over)
